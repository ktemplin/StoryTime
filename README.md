# StoryTime
A bedtime story generator using OpenAI's API to create a story for my kids (or whoever feels like it) based on defined prompts.

## Data Requirements:
 1. User Data
 2. Pre-seeded Themes for AI prompt via API
 3. Stored responses from the API for future review of previous prompts
